What is the recommended RPO for etcd?
For production clusters, Red Hat recommends an RPO of no more than 24 hours for etcd snapshots.
Can I use deduplication with PVs?
Commvault supports dedup for block-based PVs, but not for all CSI drivers.
CRONOGRAMA
Rationale: Why this order?
The Commvault Backup Plan requires frequency and retention definitions before any backup job can be configured. See documentation reference.
Rationale: Network dependency
VLAN must be trunked on the switch before NMState can create the interface on the worker nodes.
